# Module 2: The Digital Twin (Gazebo & Unity)

Welcome to Module 2 of the Digital Twin Book. This module covers the creation and simulation of digital twins using Gazebo for physics simulation and Unity for visualization.

## Overview

This module explores the concept of digital twins in robotics, combining:
- Physics simulation with Gazebo
- Visualization with Unity
- Sensor simulation for realistic data generation

## Learning Objectives

After completing this module, you will understand:
- How to set up physics simulations in Gazebo
- How to visualize robotic systems in Unity
- How to simulate sensors for realistic data generation
- How to connect Unity and Gazebo with ROS 2
